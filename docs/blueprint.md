# **App Name**: ORA - Public Feedback Platform

## Core Features:

- Unified Feed: Display a mixed feed of Government and Citizen posts, sorted by trending relevance.
- AI Summarization: Generate short, TL;DR summaries of each post using AI, highlighting key impacts and concerns, to be displayed in post cards.
- Structured Voting: Allow users to vote 'Agree,' 'Mixed,' or 'Disagree' on posts and provide reason tags (Economic, Employment, etc.) with optional short comments. AI tool can aggregate voting reasons.
- Sentiment Analysis: Analyze post sentiment and display current trends (Mostly Agree/Mixed/Mostly Disagree) within the feed.
- Filtering and Sorting: Implement filtering options by post type (Government/Citizen), trending, and topics, with advanced sorting options.
- Post Submission: Allow citizens to submit new policy proposals, which are analyzed by AI for relevance and appropriateness before posting to the feed.
- Detailed Post View: Provide a detailed view for each post, including full descriptions, media, expanded AI analysis, voting breakdown, sentiment trend charts, and an optional comments section.

## Style Guidelines:

- Primary color: Calm blue (#5DADE2) to inspire confidence and trustworthiness.
- Background color: Very light blue (#EBF4FA), providing a clean, neutral backdrop.
- Accent color: Soft purple (#A394BF), complementing the primary color while maintaining a harmonious feel.
- Body and headline font: 'PT Sans' for a clean and modern user experience.
- Use flat, minimalist icons related to policy areas (e.g., education, healthcare).
- Social-media-inspired layout with clear visual hierarchy.
- Subtle transitions for voting actions and content loading.
